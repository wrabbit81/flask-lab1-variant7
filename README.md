# Flask Color Channel Changer

Веб-приложение для лабораторной работы №1 по дисциплине "Разработка веб-сервисов для научных и прикладных задач"

## Описание
Приложение позволяет:
1. Загружать изображения
2. Менять порядок цветовых каналов (R, G, B)
3. Показывать графики распределения цветов
4. Отображать средние значения цвета по вертикали и горизонтали

## Технологии
- Python 3.8
- Flask
- Pillow (обработка изображений)
- Matplotlib (графики)
- Bootstrap 5 (интерфейс)

## Установка
```bash
pip install -r requirements.txt
python app.py
